SOLO / Hermes 连接配置

在 Hermes Agent 中添加此 MCP Server：

Hermes MCP 配置
add_mcp_server name: proxy-mcp url: /mcp transport: streamable-http

Claude Desktop 连接配置

在 claude_desktop_config.json 中添加：

claude_desktop_config.json
{ "mcpServers": { "proxy-mcp": { "url": "/mcp", "transport": "streamable-http" } } }

SDK 客户端连接 (Python)

Python 示例
from mcp import ClientSession from mcp.client.streamable_http import streamablehttp_client async def main(): async with streamablehttp_client("/mcp") as (read, write, _): async with ClientSession(read, write) as session: await session.initialize() result = await session.call_tool("proxy_health", {}) print(result)
